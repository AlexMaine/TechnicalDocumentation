Les modules SASS

Avec Sass nous pouvons apporter des fichiers .scss via la fonction @mport

Import du fichier _variables.scss

            

                /*
                J'importe mon fichier _variables.scss qui est dans mon dossier "partials". le"_" 
                devant le nom de mon fichier permet de faire comprendre à SASS que ce fichier est un "partial" 
                un fichier qui a pour vocation d'être importé. Les partials sont très utiles pour modulariser ses fichiers CSS
                */
                @import "../partials/variables"
