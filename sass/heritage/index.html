L'heritage SASS

Avec l'héritage SASS (@extend) on va pouvoir copier les propriétés d'un selecteur vers un autre selecteur.
C'est utile dans les cas de la création de classes comme .actu, .btn

            
                // style.scss

                .btn{
                    display: inline-block;
                    text-align: center;
                    border: 1px solid;
                    padding: 10px;
                }
                .danger{
                    // Nous avons les propriétés css de .btn ici;
                    @extend .btn;
                    background: red;
                }
