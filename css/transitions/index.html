Transitions & Transforms

  • "rotate" : pivoter
  • "translate" : positionner
  • "scale" : manipuler l'echelle
  • "skew" : distordre

La propriété transform va nous permettre de manipuler visuellement un élément en faisant:


            
                div{:hover
                    transform: rotate(45deg);
                }

                div:hover{
                    transform: scale(1.2) , 2;
                }

                div:hover{
                    transform: translate(10px, 20px)
                }

                div:hover{
                    transform: skew(45deg);
                }
