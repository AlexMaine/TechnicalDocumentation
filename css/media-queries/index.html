Media Queries

Les requêtes Media Queries permette de modifier l'apparence dun site en fonction du type d'appareil et de ses caracteritiques (la resolution d'ecran ou la largeur de la zone d'affichage)

Cet exemple cible les écrans de largeur inferieur ou egale à 640px 

        

            @media screen and (max-widht: 640px){
                /* Les propriétés CSS ici */
            }

            @media screen and (min-widht: 200px) and (max-widht: 640px{
                /* Les propriétés CSS ici */
            }

Cet exemple cible les écrans dont la résolution en largeur est comprise entre 200px et 640px

        

            @media screen and (min-widht: 200px) and (max-widht: 640px{
                /* Les propriétés CSS ici */
            }
