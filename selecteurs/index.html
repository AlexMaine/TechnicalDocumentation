Les Selecteurs

Les sélecteurs définissent les éléments sur lesquelles s'applique un ensemble de règles CSS.

Cet exemple montre l'imbrication et le ciblage en Css

        
          div div{
            background-color: tomato;
        }
        .c1,.c2{
            color :white;
        }
        .c2{
            background: skyblue;
        }
        .container .block{
            color: #fff;
        }

Je sélectionne l'élément qui a la classe c1 & bloc

        
        .c1.block{
            background-color: skyblue;
        }

Last-child va selectionner le Dernier Enfant de son Parent que Si et seulement Si c'est une Div

        
          div div:last-child{
            color :pink;
        }

"div:first-of-type" Va selectionner le 1er enfant du type Div de son Parent

        
          div div:first-of-type{
            color :yellow;
        }
